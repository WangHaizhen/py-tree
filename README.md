# py-tree
一个 python 的练习树。

这里是我练习 python 的地方，提供有各种用来练手的 python 小项目。

希望自己像一个 little monkey 一样 **爬上 python 大树的顶端**。

## 使用方法
1. clone 代码到本地
```
git clone https://github.com/WangHaizhen/py-tree.git
```
2. 编写对应题目代码，命名规则为 **w_项目编号.py** （开头是字母为了方便封装和调用模块） 
栗子：  
第二题代码文件 ---> w_1_1.py

3. 编写完成，push 到 github
```
git add .
git commit -m 'update w_1_1.py'
git push origin master
```

**说明**：题号为 x_x 表示还没想好题号。

### 1_1.IP地址拼接
#### 功能描述：
IP 地址由四个 0~255 的数字组成，比如：192.168.12.34，现在只给你四个数字，你需要将他们拼接起来，组成 IP 地址的形式，并输出。

要求对每个数字进行检测，每个数字的范围为 0~255，通过检测，输出拼接后的形式。

没有通过检测，输出提示 **'[Error]: cannot compose IP address.'**

#### 输入：
```

0 0 0 0
234 345 456 567
-1 -1 -1 -1
```
#### 输出：
```

0.0.0.0
[Error]: cannot compose IP address.
[Error]: cannot compose IP address.
```

### 1_2.URL地址拼接
#### 功能描述：
URL 地址又称 **Uniform Resoure Locator，即 统一资源定位器** ，是万维网上定位资源（比如html页面）的方式，它通常的组成是：

**protocol:// hostname[:port] / path /** ，即  
**协议://主机[:端口号]/路径/**
([] 表示可选)

举个栗子：
>**http://www.test.com:8000/index.html**  
**http://www.test.com:8080/login.php**  
**ftp://45.67.89.01:21/test.jpg**

我们把这个问题简化，现在有一个网络资源 **love.png** ，给你找到它的 **协议、组成主机IP的四个数字、端口号**，你需要把他们拼接成 URL 格式，并输出。

**特殊要求**：拼接 IP 地址要调用模块 w_1_1 中的功能。

#### 输入：
```
ftp 45 67 89 01 21 love.png
ftp 456 678 890 012 21 love.png
```
#### 输出：
```
ftp://45.67.89.01:21/love.png
[Error]: cannot compose IP address.
```

### 2_4.List Battle!
#### 功能描述：
快派出你的士兵，内存就是你的战场，和电脑来一场轰轰烈烈的战争吧！

派出你的 N 名士兵，你可以指定他们的武力值（**1<= 武力值 <=9**），电脑也会 **随机** 派出不同武力值的 N 名士兵和你战斗。

所有士兵将会抓对厮杀，每个人都会和对方的每个人战斗一次。

我方士兵的武力值只有 **大于** 敌方士兵的武力值才会获得一场胜利；反之，敌方获得一场胜利。

你需要按照要求的格式输出战场厮杀情况，**'o' 代表我方胜利的场次，'x' 代表我方失败的场次**。

最后统计战况，如果我方获胜次数大于敌方，那么输出 "**Congratulations! You won!**";反之，输出 "**I'm sorry you lost , try again~**"。

输入输出格式见下：

#### 输入：
**Case 1:**
```
Enter the number of competition : 5
Enter your own player (1<= x <=9) : 5 6 7 8 9
```
**Case 2:**
```
Enter the number of competition : 5
Enter your own player (1<= x <=9) : 1 2 3 4 5
```

#### 输出：
**Case 1 answer:**
```
The enemy is coming !
War outcome: 
  5 6 7 8 9 
1 o o o o o 
8 x x x x o 
8 x x x x o 
6 x x o o o 
3 o o o o o 
Congratulations! You won!
```
**Case 2 answer:**
```
The enemy is coming !
War outcome: 
  1 2 3 4 5 
4 x x x x o 
8 x x x x x 
1 x o o o o 
1 x o o o o 
9 x x x x x 
I'm sorry you lost , try again~
```
